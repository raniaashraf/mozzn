#Mozzn
Command Line Interface for mozzn APIs
##Installation

Run the following command from your terminal:

    $ gem install mozzn

##Usage
###Mozzn Commands


Commands:
  - mozzn app SUBCOMMAND   
  - mozzn auth SUBCOMMAND  
  - mozzn help COMMAND     
  - mozzn key SUBCOMMAND   
  - mozzn update           

Detailed command usage: 

    $ mozzn app


Commands:
####mozzn app console         
####mozzn app create APPNAME  
####mozzn app destroy         
####mozzn app help [COMMAND]  
####mozzn app list            
####mozzn app resources       


    $ mozzn auth


Commands:
####mozzn auth help [COMMAND]  
####mozzn auth login           
####mozzn auth register        


    $ mozzn help



####mozzn help COMMAND

    $ mozzn key

####mozzn key add

    $ mozzn update
